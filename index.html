Project Overview

The Online Bidding System was developed to simplify the communication and transaction process between sellers and buyers in an auction environment. The platform supports multiple user roles including admin, seller, and buyer, with role-specific workflows backed by a relational database.

Sellers can list products for auction, buyers can place bids and place orders after winning, and administrators can manage customer details, seller records, auction outcomes, reports, and product data. The system reduces complexity in maintaining auction activity and improves the overall product bidding experience.

Run Instructions

  1. 1
    Install Visual Studio.
  2. 2
    Clone the repository and navigate to the code folder.
  3. 3
    Open the API solution file inside the BiddingApi folder.
  4. 4
    Run update-database --verbose in Package Manager Console to create the database.
  5. 5
    Run the API project.
  6. 6
    Open the WebApp solution file inside the BiddingApp folder.
  7. 7
    Run the WebApp project.
  8. 8
    Sign up or log in to start using the system.
