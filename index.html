ProjTech - Online Bidding System

Project Overview

The project "Online Bidding System" is the software developed to simplify the communication process between the Seller and the Buyer. The software operators include the admin, the Seller, and the Buyers. The seller is responsible for selling a product on the site where other people can buy it based on their winning in the auction. This information is stored in the database along with their details.

The admin can view Customer and Seller details, Auction details, Auction winning report, and Product details. When a person places a bid on a particular product, then they are eligible to place an order for the product when they win the auction by the end of the date when the auction closes. The system can thus reduce complexity in maintaining Bidding details and ease the process of Auction for a Product. The project is developed on the ASP .Net platform and supported by an SQL database to store user-specific details.

Run Instructions

  1. Install Visual Studio.
  2. Clone the repository and navigate to the code folder.
  3. Open the API solution file in the BiddingApi folder.
  4. Use the command "update-database --verbose" in the Package Manager Console to create the database.
  5. Run the program.
  6. Open the WebApp solution file in the BiddingApp folder.
  7. Run the program.
  8. Users need to sign up and log in if they don't have an account.
